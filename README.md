# ma225
Repositório do site da disciplina MA225 — Análise de Livros e Materiais Didáticos, oferecida pela Licenciatura em Matemática do IMECC/Unicamp. O site reúne informações institucionais da disciplina, materiais de apoio, orientações pedagógicas e páginas organizadas por oferta (semestre), preservando o histórico das turmas e produções acadêmicas..
